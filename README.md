### 3. 升级 ChatGPT Plus

#### 3.1 登录 ChatGPT 账号
1. 打开 [ChatGPT 官网](https://chat.openai.com/)
2. 使用已注册的账号登录。

#### 3.2 选择 Plus 会员订阅
1. 在 ChatGPT 界面左下角点击“升级 Plus”。
2. 按照提示选择订阅计划，当前为 $20/月。

#### 3.3 输入虚拟信用卡信息
1. 在支付页面，填写 WildCard 卡片信息。
2. 随机填写美国地址（可用地址生成器生成）。
3. 确保网络节点为美国、新加坡或欧洲，否则支付可能失败。
